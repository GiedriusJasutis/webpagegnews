### Installing packages

- server (node)
  `npm i`

- client (react)
  `cd client && npm i`

### create .env and add

<!-- app port -->

PORT=

<!-- mongodb username -->

DB_USERNAME=

<!-- mongodb password -->

DB_PASSWORD=

<!-- mongodb database name -->

DB_DATABASE=

<!-- gnews api token -->

API_TOKEN=

### Run app

'concurrently runs both server and client'
`npm run dev`

### Build client

`cd client && npm run build`
